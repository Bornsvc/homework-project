1. Arduino Uno Board




![](./pic/1.jpg)





ບອດ microcontroller ໃຊ້ສຳລັບຄວບຄຸມອຸປະກອນຕ່າງໆ, ປະມວນຜົນສັນຍານ sensor ແລະສັ່ງງານອຸປະກອນ output.

ການໃຊ້ງານ / ການເຮັດວຽກ:
ຮັບ input ຈາກ sensor, ປະມວນຜົນ, ແລ້ວສົ່ງ output ໄປຫາ LEDs, motors, ເລີ່ມການສື່ສານ, ຯล.

ຕົວຢ່າງການນຳໃຊ້:
ຄວບຄຸມໄຟອັດຕະໂນມັດແມ່ນເວລາມີຄົນເດີນຜ່ານ.

2. Breadboard




![](./pic/2.jpg)





ມັນແມ່ນຫຍັງ:
ກະດານທົດລອງຕໍ່ວົງຈອນບໍ່ໃຊ້ການ solder.

ການໃຊ້ງານ:
ໃຫ້ເຊື່ອມອຸປະກອນດ້ວຍ jumper wires ຕ່າງໆ.

ຕົວຢ່າງ:
ຕໍ່ວົງຈອນ LED + resistor + button.

3. USB Cable




![](./pic/3.jpg)





ມັນແມ່ນຫຍັງ:
ສາຍເຊື່ອມ Arduino ຫາ computer.

ໜ້າທີ່:
Upload code + ຈ່າຍໄຟ 5V.

ຕົວຢ່າງ:

4–6. Jumper Wires (M2M, M2F, F2F)




![](./pic/4.jpg)
![](./pic/5.jpg)
![](./pic/6.jpg)





ມັນແມ່ນຫຍັງ:
ສາຍເຊື່ອມຕໍ່ໃນ Breadboard ແລະໄປຫາ modules.
