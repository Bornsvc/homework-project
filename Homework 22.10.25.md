1. Arduino Uno Board
![Network Diagram](./images/Screenshot_1.png)
ມັນແມ່ນຫຍັງ:
ບອດ microcontroller ໃຊ້ສຳລັບຄວບຄຸມອຸປະກອນຕ່າງໆ, ປະມວນຜົນສັນຍານ sensor ແລະສັ່ງງານອຸປະກອນ output.

ການໃຊ້ງານ / ການເຮັດວຽກ:
ຮັບ input ຈາກ sensor, ປະມວນຜົນ, ແລ້ວສົ່ງ output ໄປຫາ LEDs, motors, ເລີ່ມການສື່ສານ, ຯล.

ຕົວຢ່າງການນຳໃຊ້:
ຄວບຄຸມໄຟອັດຕະໂນມັດແມ່ນເວລາມີຄົນເດີນຜ່ານ.

2. Breadboard

ມັນແມ່ນຫຍັງ:
ກະດານທົດລອງຕໍ່ວົງຈອນບໍ່ໃຊ້ການ solder.

ການໃຊ້ງານ:
ໃຫ້ເຊື່ອມອຸປະກອນດ້ວຍ jumper wires ຕ່າງໆ.

ຕົວຢ່າງ:
ຕໍ່ວົງຈອນ LED + resistor + button.

3. USB Cable

ມັນແມ່ນຫຍັງ:
ສາຍເຊື່ອມ Arduino ຫາ computer.

ໜ້າທີ່:
Upload code + ຈ່າຍໄຟ 5V.

ຕົວຢ່າງ:
Upload code blink LED.

4–6. Jumper Wires (M2M, M2F, F2F)

ມັນແມ່ນຫຍັງ:
ສາຍເຊື່ອມຕໍ່ໃນ Breadboard ແລະໄປຫາ modules.

ໃຊ້ແນວໃດ:
M2M → board ↔ breadboard
M2F → pin header ↔ male module
F2F → module-to-module

ຕົວຢ່າງ:
ເຊື່ອມ sensor ໄປ Arduino.

7. 9V Battery Connector

ໃຊ້ເພື່ອຈ່າຍໄຟໃຫ້ Arduino ເມື່ອບໍ່ໃຊ້ USB.

8. LEDs (Red, Yellow, Blue, RGB)

ໜ້າທີ່:
ສະແດງແສງສີຕ່າງໆ.

ການໃຊ້:
ຕ້ອງໃສ່ resistor ປ້ອງກັນ.

ຕົວຢ່າງ:
ໄຟแจ້ງເຕືອນ.

9. RGB Module

ໄຟ LED ໜຶ່ງອັນທີ່ປ່ຽນໄດ້ 16M colors.

ຕົວຢ່າງ: ໄຟປ່ອງບ້ານປ່ຽນສີ.

10. Resistors

ລົດປະລິມານໄຟຟ້າ.
ໃຊ້ປ້ອງກັນ LED ຫຼືເປັນ pull-up ສຳລັບ buttons.

11. Push Buttons

ໃຊ້ເພື່ອຮັບ input ຈາກຜູ້ໃຊ້.

ຕົວຢ່າງ: ເຮັດປຸ່ມເປີດປິດ LED.

12. Potentiometer

ຕົວຕ້ານປັບໄດ້.
ໃຊ້ປັບຄ່າ input (0–1023).

ຕົວຢ່າງ: ປັບຄວາມສະຫວ່າງ LED.

13. Active Buzzer

ສົ່ງສຽງໄດ້ທັນທີ (ມີວົງຈອນຂ້າງໃນ).

14. Passive Buzzer

ຕ້ອງສົ່ງ tone ຈາກ Arduino.

ຕົວຢ່າງ: ສຽງ Melody.

15. LCD 16x2

ເປັນຈໍຂະໜາດຂອງຂໍ້ຄວາມ 2 ແຖວຄືນ 16 ຕົວ.

16. I2C Adapter for LCD

ໃຫ້ຈໍ LCD ໃຊ້ສາຍເຊື່ອມພຽງ SDA/SCL.

17. 7-Segment Display

ສະແດງໂຕເລກ 0–9.

18. 4-Digit 7-Segment

ສະແດງເວລາ, Timer.

19. 8x8 Dot Matrix

ສະແດງຂໍ້ຄວາມ / animation ງ່າຍໆ.

20. DHT11 Sensor

ວັດອຸນຫະພູມ + ຄວາມຊຸ່ມ.

21. LM35

ວັດອຸນຫະພູມທີ່ແມ່ນແທ້ (ຄ່າເທັ່ງ).

22. Tilt Sensor

ເຊັນເຊີການຂຽວ / ກົດ.

23. LDR (Light Sensor)

ວັດແສງ.

24. PIR Sensor

ກວດຈັບຄົນເຄື່ອນໄຫວ.

25. Ultrasonic Sensor

ວັດລະຍະຫ່າງ.

26. Sound Sensor

ກວດສຽງຕົກແຕກ.

27. Water Sensor

ກວດນໍ້າ / ຄວາມຊຸ່ມ.

28. Flame Sensor

ກວດແສງໄຟ.

29. RFID Module

ອ່ານບັດ RFID 13.56MHz.

30. RFID Card / Tag

ໃຊ້ເປັນຕົວຕົນປະຈຳ RFID.

31. IR Receiver

ຮັບສັນຍາລ IR ຈາກ remote.

32. IR Remote

ສົ່ງຄຳສັ່ງຜ່ານອິນຟາເຣດ.

33. Joystick Module

ຈໍສະແໜງ X/Y + ປຸ່ມກົດ.

34. 4x4 Keypad

ປຸ່ມຕົວເລກ 0–9 ແລະ A-D.

35. Relay Module

ຄວບຄຸມໄຟ 220V ຫຼື motor.

36. Servo Motor

ມັນຄວບຄຸມມຸມ 0–180°.

37. Stepper Motor

ຄວາມແມ່ນຍຳສູງ, ຂຽນເຄື່ອນເປັນຂັ້ນ.

38. Stepper Driver Module

ໃຊ້ຄວບຄຸມຂັ້ນມືອງ motor.

39. DS1302 RTC Module

ເກັບເວລາແມ່ນແມ່ນຄົງທີ່.

40. 74HC595 Shift Register

ເພີ່ມ Output pins ຈາກ Arduino.
